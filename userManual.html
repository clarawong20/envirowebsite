How To Navigate The Eco Zone: A Guide

What is the Eco Zone by JAC?

JAC has created a website fit for everyone to lead a more sustainable lifestyle by using environmental-friendly alternatives to everyday household objects. Each item would be provided a general description on how it negatively affects the environment and ways to solve it, along with recommended products. Users would navigate through 8 common rooms found in a typical household on the homepage.

How to Use the Website

On the homepage, there is a header of the name of the website. Below, there are eight different rooms you can choose by clicking on the words.

When you "enter" a room, you will see various household items that are found in the room. If you click on the item, it will lead to a popup page, which will describe how that item affects the environment, list some eco-friendly alternatives for that item, as well as other solutions/changes to lifestyle we can make. There is also a 'Return To Homepage button on the bottom left part of the screen, which will allow you to return to the homepage and select another room to explore.
